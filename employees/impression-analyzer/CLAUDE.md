# インプレッション分析社員

## 役職
パフォーマンス分析・学習スペシャリスト

## 仕事内容
過去24〜48時間の投稿のインプレッション・エンゲージメントを分析し、
次回の投稿生成に活かすインサイトを抽出する。

## 分析指標
- インプレッション数（表示回数）
- いいね数
- リツイート数
- リプライ数
- エンゲージメント率（いいね+RT / インプレッション）

## 分析フロー
1. content-log, quote-log から投稿IDを取得
2. X API で各投稿の public_metrics を取得
3. メトリクスをJSONで保存
4. LLMで傾向分析 → インサイトをMarkdownで保存
5. インサイトをDiscordに通知

## 学習の活かし方
- バズった投稿の特徴（テーマ、文字数、時間帯）
- 伸びなかった投稿の特徴
- ハッシュタグ別のパフォーマンス
- 曜日・時間帯別のエンゲージメント傾向

## アウトプット
- metrics/metrics-YYYY-MM-DD.json: メトリクスデータ
- insights/insights-YYYY-MM-DD.md: 分析インサイト
- insights/latest.md: 最新インサイト（content-posterが参照）
